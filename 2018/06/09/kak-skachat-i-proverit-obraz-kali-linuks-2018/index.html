Как скачать и проверить образ Кали Линукс 2018

Рубрика: Загрузка

Самый свежий вариант мощного хакерского дистрибутива Kali Linux 2018.2 уже доступен на сайте разработчика. Пользователей ждут обновления инструментов для взлома и улучшения в системе. Также можно скачать версии Kali Linux 2018 с различной разрядностью, в полном или сокращенном виде через торрент.

Содержание
  1. Как скачать Kali Linux 2018
  2. Как загрузить установщик Linux
  3. Как проверить установочный образ
  4. Еще один способ получения дистрибутива (или исходников)

Как скачать Kali Linux 2018

Для загрузки последней версии нужно использовать официальный ресурс дистрибутива. Все варианты загрузки представлены на странице https://www.kali.org/downloads/ .

Для приглянувшегося образа нужно нажать на ссылку Torrent. При установленном клиенте автоматически начнется добавление торрента. Если его нет, используйте https://www.utorrent.com/ и установите μTorrent или другой клиент для загрузки.

Как загрузить установщик Linux

В μTorrent откройте скачанный торрент файл нажатием на иконку листочка с плюсом.

Затем укажите параметры сохранения Кали Линукс 2018.2 (на русском).

Если отмечена галочка о запуске, программа сразу начнет загружать последнюю версию дистрибутива с торрента.

Нужно дождаться завершения и остановить раздачу Kali Linux 2018.2.

Также советуем узнать как скачать образ Кали Линукс ISO

Как проверить установочный образ

Важно проверить готовый образ Kali Linux 2018.2 на корректность. Для этого необходимо посчитать его контрольную сумму по методу SHA-256. Разработчики предлагают использовать Hashtab –дополнение к Windows. Скачать его можно с сайта http://implbits.com/products/hashtab/ , где есть бесплатный вариант.

При запуске скачанной программы может появиться системный запрос, с которым нужно согласиться.

В самом установщике достаточно нажатием Enter’а пройти по всем шагам. Когда инсталлятор закроется, можно приступать к проверке: открыть свойства скачанного Kali Linux 2018 и перейти на вкладку с хешами.

В настройках следует добавить SHA-256.

Затем выбрать «Сравнить файл» и открыть второй файл из торрента.

При полном совпадении появится зеленый индикатор – торрент загружен верно.

Еще один способ получения дистрибутива (или исходников)

Альтернативный вариант – использовать репозитории со страницы https://docs.kali.org/general-use/kali-linux-sources-list-repositories. Активным является Kali Rolling, именно в нем ведется разработка. Обратите внимание, что самая свежая версия может работать со сбоями.

Версия Sana считается устаревшей.

Имеется даже источник для первой версии, Moto.

Есть и зеркала, их список указан по адресу https://docs.kali.org/community/kali-linux-mirrors.
