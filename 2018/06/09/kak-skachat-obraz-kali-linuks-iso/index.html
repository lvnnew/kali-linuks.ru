Как скачать образ Кали Линукс ISO

Рубрика: Загрузка

Хакеры не дремлют, так что лучше подготовиться к защите с помощью Kali Linux ISO, специализированного дистрибутива для тестирования на проникновение. Иными словами, попробовать взломать собственную защиту, чтобы устранить все уязвимости. Для подготовки потребуется скачать Kali Linux ISO или файл от виртуальной машины с уже готовым к использованию инструментом (напрямую или через торрент) для 2.0, 32, 64 bit, а также для virtualbox.

Как скачать Kali Linux ISO через торрент

Инструменты в пакете слишком мощные, чтобы слепо доверять сторонним источникам. Так что за ISO нужно идти на сайт проекта https://www.kali.org/downloads/ . И не стоит искать оригинал на русском: сайт традиционно для IT английский, русская версия входит в состав многоязычного установщика.

Из таблицы ISO выберите подходящую версию Kali Linux.

Затем сохранить торрент файл к себе, сделав правый клик по слову Torrent у верного варианта – «Сохранить как», и открыть его в торрент клиенте, чтобы скачать ISO.

Документ  Linux для Virtualbox можно скачать с ресурса разработчиков https://www.offensive-security.com/kali-linux-vm-vmware-virtualbox-hyperv-image-download/ .

Чем скачать

Когда в системе инсталлирована качалка торрентов, надо просто открыть скачанный торрент файл с Kali Linux двойным кликом. Если отсутствует – на https://www.utorrent.com/ нужно загрузить μTorrent или использовать любой другой по вкусу.

Добавить торрент можно из меню Файл или кнопкой с главной панели приложения.

После указания торрент файла в открывшемся окне необходимо выбрать место для сохранения Kali Linux и нажать Ок.

Начнется загрузка. Потребуется какое-то время, чтобы скачать ISO.

Когда процесс завершится (дойдет до 100%), раздачу стоит остановить.

Как проверить

Готовый установочный Kali Линукс (на русском) нужно проверить на отсутствие ошибок. Для этого необходимо сравнить его SHA-256 контрольную сумму с данными из текстового документа – он лежит в папке, куда был сохранен торрент.

Создатели Kali Linux рекомендуют Hashtab, но можно использовать любой аналог по своему вкусу. Рабочая версия утилиты лежит по адресу http://implbits.com/products/hashtab/ , далее выбрать Free и нажать Download. Затем запустить скачанный установщик и согласиться с:

а) запросом от Windows;

б) лицензионным соглашением (появится после нажатия Next на приветственном экране установщика);

в) местом установки;

После закрытия установщика кнопкой Finish, перейдите в папку с Kali Linux и открыть детальную информацию ISO/виртуальной машины (клик правой кнопкой – Свойства), вкладку с хешем.

Пока идет вычисление, в настройках следует добавить SHA-256.

Затем указать контрольный документ для проверки хеша Kali Linux.

В норме появится зеленая галочка.

Если ее нет – файл битый, нужно скачать ISO или образ диска с торрента заново.

Как использовать ISO

Готовый ISO-файл можно прожечь на DVD или записать на флешку для установки. Виртуальную машину нужно открыть в эмуляторе. Обратите внимание, что пароль по умолчанию toor, а SSH ключи хоста сгенерированы заранее.
