Кали Линукс 2.0: скачивание и проверка работоспособности

Рубрика: Загрузка

Безопасность можно улучшать постоянно. Особенно сильный эффект дает проверка Kali Linux 2.0: скачать его несложно, а широкий набор инструментов для проверки IT-инфраструктуры на уязвимость позволит подвергнуть свою линию обороны практически боевому испытанию. Русская версия доступна как на сайте разработчика, так и через торрент.

  2. Загрузка через торрент

Как скачать Kali Linux 2.0

Единственное место, где нужно искать, а затем скачать Kali Linux 2.0 – это https://www.kali.org , официальный сайт его разработчиков. Он интернациональный, на русском не показывается.

Несколько раз в год здесь выкладываются обновленные сборки для систем 32 бит и 64 бит. Общий список актуальных релизов (полная, облегченная и т.д.) представлен на странице загрузки. Для русской версии нет отдельной позиции, так как язык выбирается при установке.

Для перехода к скачиванию нужно выбрать пункт Загрузки (Download).

В таблице даны краткое описание версии, ссылки на загрузку, размер, версия релиза, контрольная SHA-256 сумма. Например, на рисунке ниже первая строка – полный образ 64 bit, выпущенный вторым в 2018 году (всегда показывается последняя версия).

Загрузка через торрент

Для скачивания Kali Linux 2.0 нужно использовать торрент файл – он находится по ссылке «Torrent» в соответствующей строке. Потребуется установленный клиент для скачивания (например, на https://www.utorrent.com/ имеется русский вариант популярного μTorrent).

Если ассоциация с торрентами настроена, он откроется автоматически. Если нет – нужно сохранить файл *.torrent. Затем вручную добавить Kali Linux 2.0: скачать можно только числящиеся в списке торренты.

Русская версия клиента при этом запросит указать параметры: куда положить образ Кали Линукс (на русском), запускать процесс сразу или только поместить в очередь.

Должны быть выделены оба подпункта Kali Linux 2.0: скачать *.ISO нужно для доступа к дистрибутиву, а *.sha256sum для проверки его на корректность.

Примечание: Сочетание amd64 в имени торрент файла для Kali Linux 2.0 указывает не на производителя процессора, а на архитектуру 64 bit.

Необходимо дождаться, когда состояние изменится на «Раздается» (в русском варианте) — это значит, что торрент загрузился полностью.

Теперь можно остановить торрент или же оставить раздаваться – если не жалко трафика.

Проверка контрольной суммы

Нужно убедиться, что загружен не сбойный дистрибутив Kali Linux 2.0: скачать и установить любое средство проверки. Разработчики рекомендуют скачать Hashtab. У него по адресу http://implbits.com/products/hashtab/ выложен дистрибутив на английском, но при установке в русской версии Windows сама программа будет работать с русским языком. Хватит бесплатной версии.

Если система спросит, нужно одобрить запуск установщика, затем прокликать Next – I Agree – Install — Finish. Инсталлятор закроется.

В свойствах скачанного через торрент ISO-шника (правый клик по файлу – Свойства) на вкладке с хешами потребуется нажать «Настройки и подключить SHA-256».

Он добавится в список контрольных сумм. Теперь нужно открыть файл *.sha256sum кнопкой «Сравнить файл». Зеленая галочка – все в порядке, нет её – придется повторить процесс с Kali Linux 2.0: скачать торрент заново.

Ну а затем проверить на прочность свой IT рубеж.
